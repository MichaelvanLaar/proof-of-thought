# Proof of Thought (TypeScript)

TypeScript port of [ProofOfThought](https://github.com/DebarghaG/proofofthought) by DebarghaG.

This package provides the same functionality as the original Python implementation but for the Node.js/TypeScript ecosystem.

## Credits

Based on the paper "Proof of Thought: Neurosymbolic Program Synthesis allows Robust and Interpretable Reasoning" and the original Python implementation by DebarghaG.
