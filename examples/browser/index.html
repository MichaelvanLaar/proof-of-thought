🧠 ProofOfThought Browser Example

✅ Fully Functional Browser Example! This example demonstrates proof-of-thought with Z3 WASM in the browser. To use:
  1. Make sure the browser bundle is built: npm run build
  2. Serve this directory via HTTP server (e.g., python -m http.server 8000)
  3. Navigate to http://localhost:8000/examples/browser/
  4. On first load, the page will automatically reload to enable SharedArrayBuffer support
  5. Enter your OpenAI API key and try the example!
The example uses a service worker to enable COOP/COEP headers required for SharedArrayBuffer. The WASM adapter will automatically handle theorem proving without requiring native Z3 installation.
